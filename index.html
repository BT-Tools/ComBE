Features

  • Construct trees either textually or graphically.
  • Real-time synchronisation. No need to save your files while switching between editors.
  • Useful "editor services", such as error checking and selection sharing. See ... for a complete overview.
  • Layout preservation
  • Error handling
  • Same textual syntax as TextBE; possible to use existing .bt files.

Editor Services

Selection Sharing and synchronised Save, Undo and Redo.

Textual editor services:
  • Error checking: duplicate requirement, component or state names. Unresolved references.
